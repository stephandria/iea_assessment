# Chosen task

The chosen task is Task B: display the countries in countries.json inside a searchable and scrollable input.

# How to run the app

1. Start the backend

```bash
    cd back
    npm install
    npm start
```

2. Start the frontend

```bash
    cd front
    npm install
    npm run dev
```

3. Go the frontend URL `http://localhost:5173` to display the page
